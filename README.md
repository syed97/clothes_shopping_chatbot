# Clothes Shopping Assistant Bot

## Goal
The goal of this project is to develop an MVP of an AI-powered chatbot that provides personalized clothing recommendations based on user preferences. By understanding user inputs and preferences such as height, physical build, color considerations, weather considerations, and clothing needs, the assistant will suggest suitable outfits by interfacing with a shopping database. This enhances the shopping experience by reducing search time and improving personalization. 

## Scope
We will develop a limited prototype of the chatbot with the following features:
- A natural language interface that allows users to input their clothing preferences through a structured conversation.
- Basic natural language understanding (NLU) to extract user preferences from free-text input.
- Integration with a small-scale database of shopping items (limited dataset) to fetch clothing options.
- A recommendation system that suggests clothing items based on the extracted inputs and items available.
- To keep the scope managebale, the chatbot will focus on a limited selection of product types and variety.
